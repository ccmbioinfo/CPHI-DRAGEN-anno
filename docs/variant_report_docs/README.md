# CSV report documentation

This folder contains documents that describe the CSV reports generated by `CPHI-DRAGEN-anno`.

`DRAGEN_sequence_variant_report.md` documents the following reports:
- `reports/<family>.wgs.coding.CH.hg38.csv`
- `reports/<family>.wgs.panel.CH.hg38.csv`
- `reports/<family>.wgs.panel-flank.CH.hg38.csv`
- `reports/<family>.wgs.denovo.CH.hg38.csv`
- `reports/<family>.wgs.high.impact.CH.hg38.csv`


`DRAGEN_SV_CNV_report.md` documents:
- `reports/<family>.sv.CH.hg38.csv`
- `reports/<family>.cnv.CH.hg38.csv`

`DRAGEN-STR_known_pathogenic_repeats.md` documents:
- `reports/<family>.known.path.str.loci.hg38.csv`

`DRAGEN_mitochondrial_variant_report.md` documents:
- `reports/<family>.mito.hg38.csv`

`../pipeline_docs/acmg_secondary_findings_gene_annotation.md` documents:
- `reports/<family>.ACMG.SF.hg38.csv`

`../pipeline_docs/compound_het_variant_annotation.md` documents:
- `reports/<family>.compound.het.status.CH.hg38.csv`, as well as the compound het annotations added to the variant reports that contain `CH` in the filename
